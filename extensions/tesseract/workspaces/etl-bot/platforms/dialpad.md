# Dialpad Platform Reference

## Authentication
- **Type**: API Key (api_key)
- **Required credentials**: api_key
- **Where to get them**: Dialpad Admin > Developer > API Keys

## Key API Paths (via gateway)
- `GET /users` - List users
- `GET /callrouters` - List call routers
- `GET /offices` - List offices
- `GET /recordings` - List recordings

## Credential Fields (PlatformConnection)
- `api_key` - Dialpad API key
